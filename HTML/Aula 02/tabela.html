Tags para tabelas

  • thead: Cabeçalho da tabela -> contém títulos de colunas.
  • tbody: Corpo da tabela -> onde os dados principais são armazenados.
  • tfoot: Rodapé da tabela -> usado para resumir ou fornecer informações adicionais.
  • tr: table row -> linha da tabela
  • td: table data -> célula de dados da tabela.
  • th: table header -> célula de cabeçalho da tabela
